## 🔬 核心理论知识

### 1. 傅里叶变换基础概念

#### 问题的提出
在信号处理中，我们经常需要回答这样的问题：
- 这个信号中包含哪些频率成分？
- 哪些频率的强度较大，哪些较小？
- 不同频率成分之间有什么相位关系？

傅里叶变换正是为了回答这些问题而诞生的。

#### 核心思想
**傅里叶变换的本质**：任何复杂的周期性信号都可以分解为无穷多个简谐波（正弦/余弦信号）的叠加。

用公式表述：
$$f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left[ a_n \cos(n\omega_0 t) + b_n \sin(n\omega_0 t) \right]$$

其中：
- $f(t)$ 是原始时间信号
- $\omega_0 = 2\pi f_0$ 是基频（基本频率）
- $a_n, b_n$ 是傅里叶级数的系数，代表各频率分量的幅度
- $n$ 表示谐波次数（1倍频率、2倍频率、3倍频率...）

### 2. 傅里叶级数 (Fourier Series) - 周期信号的频域表示

#### 什么是傅里叶级数？
傅里叶级数用于**周期性信号**的频域分析。它将一个周期为 $T$ 的周期信号表示为不同频率正弦波的加权和。

#### 复指数形式（更优雅的表达）
$$f(t) = \sum_{n=-\infty}^{\infty} c_n e^{j2\pi n f_0 t}$$

其中：
- $f_0 = 1/T$ 是基频（最低频率）
- $c_n$ 是复数形式的傅里叶系数，包含幅度和相位信息
- 正负n代表正负频率（虽然在实际信号中，负频率与正频率是共轭的）

#### 从系数看信号的频率成分
对于实信号，我们可以用幅度谱和相位谱来描述：
- **幅度谱**: $|C_n| = \sqrt{a_n^2 + b_n^2}$ - 表示第n个频率分量的强度
- **相位谱**: $\angle C_n = \arctan(b_n/a_n)$ - 表示第n个频率分量的初始相位

### 3. 时域 vs 频域

| 维度 | 时域 (Time Domain) | 频域 (Frequency Domain) |
|------|-------------------|----------------------|
| **轴** | 时间 t | 频率 f 或 ω |
| **含义** | 信号随时间的变化 | 信号中各频率成分的强度 |
| **数据** | 幅度随时间变化的曲线 | 离散的频率与对应的幅度 |
| **直观感受** | 看听觉信号，能听到音的高低变化 | 看频谱，能看到信号中主要有哪些音 |
| **应用** | 音频处理、图像处理、控制系统 | 频率分析、滤波、压缩 |

**关键认识**：同一个信号，时域表示和频域表示是完全等价的，只是角度不同。

### 4. 快速傅里叶变换 (FFT) 与离散傅里叶变换 (DFT)

在数字信号处理中，我们处理的是**离散时间信号**（采样信号），因此使用：

#### 离散傅里叶变换 (DFT)
$$X[k] = \sum_{n=0}^{N-1} x[n] e^{-j2\pi nk/N}$$

其中：
- $x[n]$ 是时间域的第n个采样点
- $X[k]$ 是频域的第k个频率分量
- $N$ 是采样点数

#### 快速傅里叶变换 (FFT)
FFT是计算DFT的高效算法，将计算复杂度从 $O(N^2)$ 降低到 $O(N \log N)$。

### 5. 周转圆（Epicycle）- 傅里叶级数的几何解释

这是理解傅里叶级数最直观的方法：

**想象一个复杂的运动轨迹**，可以分解为多个圆周运动的叠加：
- 第1个圆以基频 $f_0$ 旋转
- 第2个圆以 $2f_0$ 旋转（速度是第1个圆的2倍）
- 第3个圆以 $3f_0$ 旋转
- ...以此类推

每个圆的：
- **半径** = 该频率分量的幅度 $|c_n|$
- **旋转速度** = 该频率分量 $nf_0$
- **初始角度** = 该频率分量的相位 $\angle c_n$

**关键结论**：将所有圆的终点连接起来，得到的轨迹就是原始信号在时域的表示！

## 🎮 功能模块说明

### 1. 信号构建器 (Signal Builder)

允许你通过添加不同频率的正弦波来构建自己的信号。

**使用方法**：
1. 调节"频率"滑块（1-20 Hz）
2. 调节"振幅"滑块（0-1）
3. 调节"相位"滑块（0-360°）
4. 点击"➕ 添加谐波"

你也可以快速加载预设信号：
- **纯正弦波**: 单一频率的简单信号
- **方波近似**: 用多个奇数次谐波组成的方形脉冲
- **锯齿波**: 包含所有谐波的波形
- **三角波**: 仅含奇数次谐波但衰减较快
- **脉冲串**: 频域中均匀分布的频率
- **拍频现象**: 两个接近频率的信号相互干涉

### 2. 时域信号 (Time Domain)

显示信号随时间的变化。

**特点**：
- 彩色虚线：各个频率分量（谐波）
- 青色实线：所有分量的**合成信号**
- 粉色竖线：当前时刻指示器，实时跟踪动画进度
- 网格和中心线：参考标尺

**学习重点**：观察如何从简单的正弦波（时域）合成出复杂的波形。

### 3. 频谱分析 (Frequency Domain)

显示信号的频域表示。

**上半部分 - 幅度谱 (Magnitude Spectrum)**：
- 横轴：频率（Hz）
- 纵轴：该频率分量的强度
- 柱状图：清晰显示信号中的主要频率成分

**下半部分 - 相位谱 (Phase Spectrum)**（可选）：
- 显示每个频率分量的初始相位（-180° 到 +180°）
- 曲线波动反映了信号的复杂性

**可选功能**：
- ☑️ 显示相位谱：切换是否显示相位信息
- ☑️ 对数坐标：使用对数刻度强调弱信号

**学习重点**：同一个信号，频域视角显示了它由哪些"成分"组成。

### 4. 旋转矢量可视化 (Epicycles)

这是最直观的傅里叶级数几何表示，堪称"数学可视化艺术"。

**左侧 - 周转圆**：
- 多条箭头，逐个首尾相连
- 每条箭头都在旋转（旋转速度不同）
- 最终的终点（红点）随时间移动

**右侧 - 波形展开**：
- 将周转圆的轨迹"展开"为普通的時域波形
- 与左侧的时域图同步变化

**中间 - 虚线连接**：
- 连接左右两个图，展示它们的对应关系

**参数调整**：
- **动画速度**：加快或减慢旋转
- **矢量数量（精度）**：使用更多的频率分量，信号近似得更精确

**学习重点**：这是理解"傅里叶级数为什么有效"的最好方式。你会看到，即使加入20-30个频率分量，也能很好地近似相当复杂的波形！

## 🧠 学习路径建议

### 初级阶段（理解基本概念）

1. **从纯正弦波开始**
   - 加载"纯正弦波"预设
   - 观察时域中的光滑曲线
   - 观察频域中只有1个尖峰
   - 理解：最简单的信号就是单一频率

2. **改变频率和振幅**
   - 添加不同频率的正弦波
   - 观察时域如何变复杂
   - 观察频域出现多个尖峰，频率越高的分量越靠右
   - 理解：复杂信号 = 简单信号的加法

3. **观察相位的作用**
   - 添加两个同频率的信号，改变其中之一的相位
   - 观察时域波形如何移动
   - 理解：相位控制波形的"位置"

### 中级阶段（理解频域特性）

4. **研究预设信号**
   - 逐个加载不同的预设
   - 预测每个预设的频谱样式
   - 解释为什么方波需要更多高频分量
   - 理解：不同形状的信号对应不同的频谱

5. **实验奇偶函数**
   - 方波（偶函数）/ 三角波（奇函数）的对称性
   - 观察其频谱中只出现奇数次或偶数次谐波
   - 理解：信号的对称性反映在频谱中

6. **尝试对数坐标**
   - 启用"对数坐标"
   - 观察弱信号变得可见
   - 理解：对数坐标的妙用（处理幅度差异大的信号）

### 高级阶段（几何直观理解）

7. **周转圆的魔力**
   - 加载方波并查看周转圆
   - 逐渐增加矢量数量
   - 动画观察，看终点是如何描绘出方波的
   - 理解：傅里叶级数不仅是数学，更是几何

8. **预测与验证**
   - 自己手动组合几个正弦波
   - 推测结果的时域和频域样子
   - 运行动画验证
   - 理解：从数学到直观的转变

## 💡 数学公式速查表

### 基本定义

| 公式 | 含义 |
|------|------|
| $f(t) = A \sin(2\pi f t + \phi)$ | 单个正弦信号：A=振幅，f=频率，φ=相位 |
| $f(t) = \sum a_n \sin(2\pi n f_0 t)$ | 傅里叶级数：无穷多个不同频率的叠加 |
| $X[k] = \sum_{n=0}^{N-1} x[n] e^{-j2\pi nk/N}$ | 离散傅里叶变换 (DFT) |

### 关键关系

| 性质 | 说明 |
|------|------|
| 线性性 | $\mathcal{F}\{a \cdot f + b \cdot g\} = a \cdot F + b \cdot G$ |
| 时移 | 时域延迟 ↔ 频域添加线性相位 |
| 频移 | 时域乘以复指数 ↔ 频域平移 |
| 卷积定理 | 时域卷积 ↔ 频域乘法 |
| 帕塞瓦尔定理 | 时域能量 = 频域能量 |

## 🎯 常见问题解答 (FAQ)

**Q1: 为什么需要频域分析？**
A: 同样的现象在频域中更容易理解。例如：
- 音频：能直接看出音的"音色"由哪些泛音组成
- 图像：高频分量代表细节，低频代表大轮廓
- 通信：检测干扰信号的频率

**Q2: 什么时候用傅里叶级数，什么时候用傅里叶变换？**
A: 
- **傅里叶级数**：处理周期信号（如交流电、心电图）
- **傅里叶变换**：处理非周期信号或能量有限的信号（如单个脉冲、音频片段）

**Q3: 为什么要用复数（虚数）？**
A: 复数优雅地同时编码了幅度和相位信息。欧拉公式 $e^{i\theta} = \cos\theta + i\sin\theta$ 是关键。

**Q4: DFT 和 FFT 有什么区别？**
A: FFT 是计算 DFT 的快速算法，数学结果完全相同，但FFT 快得多（特别是当N很大时）。

**Q5: 如何选择合适的采样率？**
A: 根据奈奎斯特采样定理，采样率需要是最高频率分量的2倍以上，否则会发生混叠（aliasing）。
